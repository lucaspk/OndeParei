# Onde Parei?

'Onde Parei?' é uma ferramenta pensada para lembrar onde você parou em uma determinada atividade. 

Você gosta de assistir muitas séries, mas se esquece sempre em qual episódio parou ? Não tem problema, pois OndeParei vai mostrar em qual episódio você parou!

Você estava lendo um livro e acabou esquecendo em qual página parou ? Não se preocupa, pois OndeParei tem essa informação!
